# Numerical Integration for Finance (C)

Project developed at EPITA (Engineering School in Computer Science) as part of the "Probability, Insurance and Finance" module.

## Overview

This project focuses on implementing numerical integration methods to model probabilistic problems in finance.

## Objectives

* Approximate integrals using numerical methods
* Apply probability concepts to financial modeling
* Develop efficient implementations in C

## Technologies

* C (low-level programming)
* Numerical methods
* Probability theory

## Skills Developed

* Mathematical modeling
* Numerical analysis
* Algorithm implementation in C
* Understanding of probabilistic systems

## Context

This project was completed as part of the EPITA curriculum, emphasizing both theoretical understanding and practical implementation of mathematical concepts in computer science.
